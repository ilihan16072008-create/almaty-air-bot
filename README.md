# 🌫 Almaty Air Bot

Telegram-бот который показывает качество воздуха в Алматы в реальном времени.

##  Возможности

 - Показывает актуальный AQI по Алматы
 - Даёт советы в зависимости от уровня загрязнения
 - Утренние уведомления каждый день в 8:00
 🟢🟡🟠🔴 Цветовая индикация уровня опасности

##  Команды бота

| Команда     | Описание |
|-------------|----------|
| /start      | Начать работу с ботом               |
| /air        | Узнать AQI прямо сейчас             |
| /subscribe  | Подписаться на утренние уведомления |
| /unsubscribe| Отписаться от уведомлений           |

##  Технологии

- Python 3.13
- python-telegram-bot
- IQAir API
- APScheduler

##  Установка

1. Клонируйте репозиторий
2. Создайте файл `.env` по примеру `.env.example`
3. Вставьте свои ключи в `.env`
4. Установите библиотеки:
pip install python-telegram-bot requests apscheduler python-dotenv
5. Запустите бота:
python bot.py
